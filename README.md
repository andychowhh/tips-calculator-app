[**Try out the demo!**](https://tips-calculator-app-andychowhh.vercel.app/)

## Build and run

### To Run Locally:

Clone the repository and perform the following command line actions:

```bash
$> npm install
$> npm run dev
```

Then open http://localhost:3000 in browser.

### To Build:

```bash
$> npm run build
```

## Testing

### Storybook

```bash
$> npm run storybook
```
